# 🤖 User Management System з Telegram-ботом

Це Spring Boot застосунок для управління користувачами з інтеграцією Telegram-бота. На даний момент бот відображає створеного користувача через повідомлення.

## 🔧 Налаштування проєкту

### 1. Клонуйте репозиторій

### 2. Створіть Telegram-бота

1. Відкрийте @BotFather у Telegram

2. Введіть команду /newbot

3. Вкажіть ім'я та username бота (наприклад, UserBot)

4. Скопіюйте токен, який видасть BotFather — виглядає приблизно так: 123456789:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

### 3. Налаштуйте application.properties

Використовуйте файл application.example.properties як шаблон

1. Створіть файл src/main/resources/application.properties <-- (Цей файл додано в .gitignore, щоб не потрапляв у публічний репозиторій.)

2. Вставте свої значення

### 4. Запуск застосунку

Через термінал: ./mvnw spring-boot:run

Або в IDE:
Запустіть головний клас: UserManagementSystemApplication.java
